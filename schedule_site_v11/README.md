# Schedule site (режим/план/факт по 30 минут)

## Запуск
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

Откройте http://localhost:8000

## Как работает
- Загружаете Excel выгрузку "Заявки базовый ..."
- Выбираете дату.
- Фильтры: введите подстроку для любого столбца из выгрузки.
- Режим и Факт: кликайте по ячейкам (по 30 минут) — значения сохраняются в SQLite (data/app.sqlite3).
- План: строится автоматически по "Дата подачи" + "Время подачи" → "Время завершения" и группируется по "Гос номер итогового ТС" (если нет — "Гос номер ТС").
- Экспорт: кнопка "Выгрузить Excel" отдаёт файл export_YYYY-MM-DD.xlsx с заливками.

## Настройка полей
Если в вашей выгрузке называются иначе колонки времени/номера — поменяйте в `compute_plan()` и формировании списка vehicle.
